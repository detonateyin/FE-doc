# 浏览器之跨域问题

1. 何时发生跨域
    1. 协议不同
    2. 子域不同
    3. 端口不同
    4. IP不同

2. 为何跨域不被允许？
   
   浏览器出于安全考虑的同源策略

3. 跨域解决办法
    1. Access-Control-Allow-Origin
    2. 代理
      1. Nginx
      2. http-proxy-middleware
    3. postMessage
    4. JSONP,即利用script可跨域的特性完成跨域请求
    5. window.name
    6. location.hash
    7. window.domain
    8. websocket
